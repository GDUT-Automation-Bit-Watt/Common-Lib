# common-lib

Shared libraries and engineering utilities for GDUT Automation Bit&Watt Team.

## Purpose

This repository stores reusable modules and common engineering resources used across different student projects and competitions.

The goal is to avoid duplicated code and establish a unified engineering framework.



## Team Members

## Contents

- UART communication
- CAN communication
- PID controller
- Filters
- Motor control utilities
- Sensor drivers
- Debug tools
- Common protocol definitions

## Rules

- Keep modules independent and reusable
- Add documentation for every new module
- Test before merging into main branch
- Avoid project-specific hardcoded logic
